How It Works

  1. Register your API endpoint using a cURL command or by filling the form.
  2. We validate and generate an MCP-compatible tool interface.
  3. Tools become discoverable via the Model Context Protocol.
  4. Monitor usage via the analytics dashboard.

How to Use MCP Tools

Copy the MCP config and paste it into compatible tools like Cursor or Windsurf.

Select a tool to view its configuration.

To integrate, use the "View Config" button in the Registered Tools tab.
